# 야르한끼 브랜드 홈페이지

냉면·돈까스 세트 전문 브랜드 "야르한끼"의 브랜드 홈페이지입니다.
HTML5 · CSS3 · Vanilla JavaScript로만 제작했습니다 (프레임워크 미사용).

## 폴더 구조

```
yareun-hanki/
├── index.html          메인 페이지
├── css/style.css        전체 스타일 (CSS 변수 기반 디자인 시스템)
├── js/main.js            인터랙션 (헤더, 탭, 아코디언, 슬라이더, 스크롤 애니메이션)
├── icons/mark.svg        브랜드 마크(로고 아이콘) - 기획서의 원형 접시·Y자 면·젓가락 컨셉을 벡터로 제작
├── images/                현재 모두 "임시 플레이스홀더" 이미지입니다
└── videos/                히어로 영상 자리 (현재 비어 있음)
```

## 지금 바로 확인하는 방법
`index.html` 파일을 더블클릭하거나 브라우저로 열면 바로 확인할 수 있습니다.
(서버 없이도 상대경로만으로 정상 작동하도록 만들었습니다.)

## 실제 사진/영상으로 교체하는 방법

`images/` 폴더 안의 파일들은 전부 브랜드 컬러로 만든 **임시 플레이스홀더**입니다.
아래 파일명과 **동일한 이름**으로 실제 사진을 교체해 넣으시면 코드 수정 없이 바로 반영됩니다.

| 파일명 | 용도 | 권장 비율/사이즈 |
|---|---|---|
| hero-bg.jpg | 히어로 배경 이미지 (영상 없을 때 대체) | 1920x1080, 가로형 |
| hero-poster.jpg | 히어로 영상 썸네일 | 1920x1080 |
| brand-story.jpg | 브랜드 소개 섹션 (주방/셰프 컷 추천) | 세로형 4:5 |
| menu-set.jpg | 냉면·돈까스 세트 (대표 메뉴 큰 카드) | 세로형 |
| menu-naengmyeon.jpg | 냉면 | 4:3 |
| menu-donkatsu.jpg | 돈까스 | 4:3 |
| menu-mandu.jpg | 만두 | 정방형 |
| menu-drink.jpg | 음료 | 정방형 |
| signature-1.jpg | 시그니처 스토리 섹션 큰 이미지 | 가로형 |
| gallery-1.jpg ~ gallery-6.jpg | 갤러리 매소너리 그리드 | 세로/가로 혼합 |
| review-avatar-1.jpg ~ 4.jpg | 후기 작성자 프로필 (원형으로 크롭됨) | 정방형 200x200 |
| map-placeholder.jpg | 지도 자리 (실제로는 카카오/네이버 지도 API 삽입 권장) | 가로형 |
| og-image.jpg | 카카오톡/SNS 공유 시 보이는 썸네일 | 1200x630 |

**영상 추가하기**: `videos/hero.mp4` 파일을 넣으면 히어로 섹션에서 자동으로 재생됩니다.
파일이 없으면 JavaScript가 자동으로 `hero-bg.jpg` 이미지로 대체합니다 (에러 없음).

## 실제 오픈 전 꼭 채워야 할 정보
- `index.html`의 "오시는 길" 섹션: 매장 주소, 전화번호, 영업시간
- Footer의 대표자명, 사업자등록번호
- `#visit`, `.info-list` 안의 지도는 실제 카카오맵/네이버지도 API 연동 권장
- SNS 링크 (`.sns-row a`)의 `href="#"`를 실제 계정 주소로 교체
- 상표 등록 가능 여부 확인 (기획서에서 언급된 사항)

## 커스터마이징
`css/style.css` 상단 `:root` 안의 CSS 변수만 바꾸면 전체 톤을 손쉽게 조정할 수 있습니다.
예: `--color-orange` 값을 바꾸면 포인트 컬러 전체가 함께 바뀝니다.
